# Expense Tracker

A personal expense tracking application to manage daily expenses,
analyze spending patterns, and improve savings.

## Tech Stack
- Backend: Python (Flask)
- Database: MySQL/PostgreSQL
- Frontend: React (planned)
- Cloud: AWS (planned)

## Features
- JWT-based authentication
- Expense CRUD operations
- Category management
- Monthly & category-wise analytics
- Budget limits & overspending alerts
- Pagination & input validation

## Status
Backend completed. Frontend & deployment in progress.
